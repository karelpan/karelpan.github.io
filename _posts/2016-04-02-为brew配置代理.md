首先你需要有一个已经配置好的代理，比如XXNET，可自行于github上搜索有详细教程。
然后检查你的用户主目录下是否有 .curl 文件

```bash
ls ~/.curl
```

如果没有，创建它

```bash
touch -f .curl
```

添加如下信息（我假设你用的就是XXNET代理，那么默认的GAE的代理端口就是8087）

```bash
#HTTP代理
proxy=127.0.0.1:8087
```
